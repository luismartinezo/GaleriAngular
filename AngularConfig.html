ANGULAR

  • INSTALAMOS ANGULAR GLOBAL
  • CREAMOS EL PROYECTO ANGULAR CON ng new nombre_proyecto
  • CREAMOS LA APLICACION CON ng serve
  • CREACION DE COMPONENTES CON ng generate component nombre_componente o resumido ng g c nombre_componente(tener cuidado con los nombres que no sean iguales).
  • Podemos tambien clonar un QuickStart con git clone https://github.com/angular/quickstart nombre_proyecto (Intslamos las dependencias con npm install dentro del proyecto).
  • CREAMOS UN PROYECTO PARA LA GALERIA DE IMAGENES CON ng new Galeria --prefix galeria (con prefix le indicamos que este sera el nombre principal en ves de App)
  • AGREGAMOS LOS CDN DE BOOTSTRAP AL INDEX
  • CREAMOMS LA BARRA DE NAVEGACION CON ng g c navegacion -is --flat (el -is --flat es para omitir el archivo component.css ) podemos ver las ayudas con ng generate component --help
  • CREAMOS UN COMPONENTE DE Galeria Y DENTRO OTRO DE Lista-imagen Y DENTRO UNO DE Imagen
  • CREAMOS CARPETA LLAMADA Modelo Y DENTRO DE ESTA CORREMOS ng g cl image PARA CREAR UNA CLASE DEL MODELO
